# basic-express-server

This is a basic Espress server that includes server modularization, use of middleware, and tests.

## Installation

* `npm install`

Set your PORT environment with an .env file

```text
PORT=3001
```

## Usage

Send a request:

```text
method: GET
route: /person
params:
  name{String}
```
